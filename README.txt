===========================================
 파일 암호화 / 복호화 유틸리티 사용 설명서
===========================================

[기능 요약]
-----------
- 이 프로그램은 특정 폴더 안의 모든 파일을 AES 방식으로 암호화하거나 복호화합니다.
- 파일 이름은 Base64 방식으로 인코딩되며 `.log` 확장자가 붙습니다.

[사용 방법]
-----------
1. `runCrypto.bat` 파일을 더블클릭합니다.

2. 아래 중 원하는 작업을 선택합니다:
   - 1 : 암호화
   - 2 : 복호화

3. 원본 파일이 들어있는 폴더 경로를 입력합니다.
   예시: `C:\dev\test\ORIGINAL`

4. 결과 파일을 저장할 폴더 경로를 입력합니다.
   예시: `C:\dev\test\ENC`

5. 작업이 완료되면 "Done." 메시지가 출력되고, 아무 키나 누르면 창이 닫힙니다.

[주의 사항]
-----------
- 이 프로그램을 실행하려면 Java가 설치되어 있어야 합니다.
  확인 방법: 명령어 창(cmd)에서 `java -version` 입력

- 파일 이름이 자동으로 인코딩되므로 원래 이름으로 복원하려면 반드시 이 프로그램으로 복호화해야 합니다.

- 복호화 시 기존에 동일한 파일명이 있는 경우 덮어쓰기 되므로 주의하세요.
